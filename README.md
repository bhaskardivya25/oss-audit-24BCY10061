## Scripts Overview

### Script 1 — System Identity Report
**File:** `script1_system_identity.sh`  
Displays current user, shell details, system information, date/time, and working directory using Git Bash commands.

---

### Script 2 — FOSS Package Inspector
**File:** `script2_package_inspector.sh`  
Checks if Python exists in Git Bash, prints version, executable path, and a short description using a case statement.

---

### Script 3 — Disk and Permission Auditor
**File:** `disk_permission_auditor.sh`  
Loops through safe directories accessible from Windows via Git Bash and reports their permissions and size.

---

### Script 4 — Log File Analyzer
**File:** `log_file_analyzer.sh`  
Creates and analyzes a sample log file, counts keyword matches, and prints recent occurrences.

---

### Script 5 — Open Source Thought Recorder
**File:** `open_source_manifesto.sh`  
Asks user questions and generates a personalized open-source note saved to a text file.
